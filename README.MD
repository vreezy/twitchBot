# Getting Started Twitch Bot
docu: https://dev.twitch.tv/docs/irc

I created a simple Bot with the BOT getStarted Tutorial from Twitch

1. git clone https://github.com/vreezy/twitchBot.git
2. npm install
3. rename .env.example -> .env
4. edit the .env
5. node bot.js

have fun!
